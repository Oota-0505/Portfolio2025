# Portfolio2025 - 10月版

## 概要
Webデザイナー・フロントエンドエンジニアのポートフォリオサイトです。
2025年10月時点での制作実績をまとめています。

## 主な制作実績

### メディパ様（美容医療）サービス全般制作
- **概要**: Googleのビジネスコンテストで優勝した医学生チームのサービス
- **技術**: WordPress、ACF、PHP、HTML/CSS/JavaScript
- **担当**: デザイン以外のフロントエンド・バックエンド全般

### VEIN ENERGY様 コーポレートサイト
- **概要**: ロゴ制作から実装まで3日で納品
- **技術**: Studio、クラウドワークス
- **特徴**: シンプルかつ明瞭なデザインで最小限の情報でも訴求

### 串かつ さじろう様 リニューアル
- **概要**: 全6ページをフルスタック〜CMS化まで実装
- **技術**: フルスタック開発

### HEART&BRAIN様 LP制作
- **概要**: パララックス効果で奥行きのある流行のデザイン
- **技術**: HTML/CSS/JavaScript

### ぎふ就労支援センター様 リニューアル
- **概要**: 大規模リニューアル案件
- **技術**: フルスタック開発

### Coffee コーポレートサイト
- **概要**: Reactを使用したモダンなコーポレートサイト
- **技術**: React、モダンなWeb技術

## 技術スタック
- **フロントエンド**: HTML5, CSS3, JavaScript, React
- **バックエンド**: PHP, WordPress
- **CMS**: WordPress（オリジナルテーマ構築）、ACF
- **その他**: Studio、レスポンシブデザイン、SEO対策

## 特徴
- レスポンシブ対応（SP/PC）
- SEO配慮
- パフォーマンス最適化
- ユーザビリティを重視した設計
